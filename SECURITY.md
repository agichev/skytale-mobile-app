# Security Policy

## Scope

This repository contains the public Android client source only. Backend infrastructure, production deployment details, and operational server code are out of scope for this publication.

## Reporting

If you identify a security issue, please report it privately to the rightsholder before public disclosure.

Recommended report contents:

- affected component and version
- impact summary
- proof of concept or reproduction steps
- suggested remediation, if available

## Allowed Research

Independent review, static analysis, reverse engineering for interoperability or security evaluation, and responsible vulnerability research are permitted under the license terms for this repository.

## Disclosure Expectations

Please allow reasonable time for triage and remediation before broad public disclosure of unpatched issues.
